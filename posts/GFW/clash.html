                          # 科学上网

                          本文档核心诉求就一个：解决网络不通的问题。 (俗称：科学上网、梯子、加速器、VPN、翻墙 …)

                          1. 互联网非法外之地，请科学上网！学做鲁迅笔下的吃瓜群众: 勿评论、勿点赞、勿转发！

                          2. 勿参与搭建机场服务，代理销售等牟利行为，仅个人科学上网暂不违法的(对于公民: 法无禁止即自由, 对于政府: 法无授权即禁止)

                          # 1. 上网配置步骤

                          1、下载、安装对应系统的clash软件。

                          (window\macbook\android\ios\linux\TV\路由器等系统)

                          2、复制有效的订阅链接。

                          (自行注册购买、借用他人的订阅链接、或者网上搜索免费共享的)

                          3、参考网上视频一分钟内可配置好clash软件

                          (也可自行搜索图文配置教程)

                          # 2. 安卓系统

                          # 6. 答疑(QA)

                          老司机请绕路，仅供刚入门科学上网的参考

                          # 6.1. 1、配置注意点

                          • 代理模式：规则模式（rule）。国内网站访问更快，无需绕圈到国外。
                          • 自动更新：配置自动更新节点。避免一段时间后节点失效，需要手动更新。

                          # 6.2. 2、clash和其他科学上网软件差异

                          对比事项通用软件定制软件
                          软件参考clash
                          v2ray
                          shadowrocket(ios专用付费)
                          Stash(ios专用付费)                          		各种加速器软件
                          各种vpn软件
                          安全参考clash、v2ray是免费通用软件
                          代码完全开源、安全有保障                          		具体看定制软件索要哪些权限
                          软件后台实际的数据收集行为
                          便利参考通用软件只提供科学上网能力
                          订阅链接需自行购买配置
                          机场跑路后无需更换软件                          		软件内置了订阅链接购买选项
                          无需额外配置，直接启动使用
                          跑路后需要更换软件
                          费用参考相对便宜（10+/月）
                          机场间竞争激烈                          		相对贵一点
                          竞争不足

                          # 6.3. 3、电视、手表等小众设备解决方案

                          使用软路由(相比于普通的路由器，可自由安装软件)解决所有接入设备的网络问题

                          上网方式路由器手机、电脑等软件
                          配置对比购买软路由
                          软路由上统一配置科学上网                          		每个设备都需要安装对应软件
                          不同设备的软件都需要配置
                          每次使用时启动
                          易用便利会连接WiFi即可
                          所有能联网的设备(电视\手表\音箱)                          		手机、电脑等设备都要安装配置
                          智能手表、音箱等没有对应app
                          费用参考路由器100+元(一次性)
                          10+/月订阅费(可多终端共享)                          		10+/月订阅费(多终端共享)
                          扩展功能广告屏蔽(手机、平板等广告过滤)
                          解锁网易云灰色歌曲
                          其他openwrt插件                          		不支持

                          # 6.4. 4、机场自费购买订阅链接

                          TIP1：建议不要购买太长时间，毕竟建机场是不受法律保护的，随时可能跑路。

                          TIP2：非工作室、视频等专业户，选择最便宜的既可以。日常搜索、图文浏览花不了多少流量。

                          TIP3：相比于网上免费共享的，付费的体验更好：高速、稳定、省心、无需频繁更换配置链接。
